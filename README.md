# Wartales 한글 패치 (Custom Edition)

이 저장소는 [DCInside Wartales 갤러리](https://gall.dcinside.com/mgallery/board/view/?id=wartales&no=31502)에서 제공된 **한글 패치**를 기반으로,  
개인적으로 커스텀을 가하여 수정/보완한 버전을 공유하기 위해 만들어졌습니다.

---

## 📌 주요 변경 사항
- 원본 패치의 번역 일부를 개인 취향에 맞게 수정

---

## 🚀 설치 방법
1. 저장소를 클론하거나 ZIP 파일로 다운로드합니다.
   ```bash
   git clone https://github.com/BoomMinho/Wartales-Korean.git
   ```
2. 압축을 해제한 후, 제공된 패치 파일을 게임 설치 경로(`res.pak` 파일이 있는 폴더)에 덮어씌웁니다. 
3. 게임을 실행하여 한글 패치가 정상적으로 적용되었는지 확인합니다.

---

## ⚠️ 주의사항
- 본 패치는 **비공식** 커스텀 버전입니다.  
- 원본 패치 제작자에게 모든 공을 돌리며, 본 저장소는 개인적인 커스터마이징 공유 목적입니다.  
- 버그나 오역이 있을 수 있으며, 자유롭게 수정하여 사용하셔도 됩니다.
